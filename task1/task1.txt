Файл с данными

В файле “task_1_capital.txt” содержатся данные об основной и арендной стоимостях 96 домов. Данные представлены в виде двух колонок: Capital – основная стоимость и Rental – арендная стоимость.

Задание

Необходимо, воспользовавшись формулой на стр. 13 слайдов второй лекции, построить линейную регрессию Rental по Capital. Входные данные отобра- зить на двумерном рисунке, на нем же нарисовать получившуюся линию регрессии. Для учета константы вектор признаков нужно дополнить единицами.

Для решения задания выбирается любой язык программирования. Можно использовать функцию для получения псевдообратной матрицы из библиотек линейной алгебры. Запрещается использование готовых методов, реа- лизующих регрессию.

Содержание ответа

Для получения зачета по этому заданию необходимо прислать следующее:

* График с данными и линией регрессии.
* Краткое описание полученного результата в контексте данных.
* Код всей программы, реализующий нахождение параметров линейной регрессии.
